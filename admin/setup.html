Настройка (один раз)

Только для того, кто заливает сайт на GitHub

← К админке
  1. Создать Google Таблицу
  2. Расширения → Apps Script
  3. Скопировать код из файла admin/apps-script/Code.gs в редактор
  4. Выбрать функцию setupOnce → Выполнить → разрешить доступ
  5. ⚙️ Свойства проекта → Script properties → ADMIN_PASSWORD = пароль для админа
  6. Развернуть → Веб-приложение → доступ «Все» → скопировать URL
  7. Вставить URL в js/listings-config.js в поле apiUrl
  8. Залить сайт на GitHub
