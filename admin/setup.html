Проверяем Google API…

1. Таблица + код

  1. Новая Google Таблица
  2. Расширения → Apps Script (только из таблицы!)
  3. Вставить весь код из admin/apps-script/Code.gs
  4. upgradeOnce → Выполнить
  5. setPasswordOnce → Выполнить
  6. testWriteOnce → Выполнить (строка в таблице)

2. Развёртывание (главное!)

Сохранить код (Ctrl+S) недостаточно. Нужно обновить развёртывание:

  1. Apps Script → Развернуть → Управление развёртываниями
  2. У активного веб-приложения нажать ✏️ (карандаш)
  3. Версия: Новая версия
  4. Развернуть
  5. Если создали новое развёртывание — скопируйте новый URL в config.js и js/listings-config.js

3. Проверка

Откройте в браузере (ваш apiUrl + ?action=ping):

Правильно: {"ok":true,"version":3,...}

Неправильно: [] — старый скрипт, повторите шаг 2
